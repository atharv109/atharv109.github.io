Projects

VulnSwarm-VEX

A research prototype for dependency vulnerability triage. The goal is to combine vulnerability intelligence with application-level evidence so teams can separate “package is vulnerable” from “this app is actually affected.”

Evidence includes dependency files, OSV/GHSA/NVD advisories, imports, callsites, reachable risky usage, source/sink signals, and guard conditions. The output is a VEX-style verdict with justification.

verdict = affected if advisory ∧ reachable_usage ∧ no_guard else under_investigation
