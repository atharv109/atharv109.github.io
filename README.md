## üé® Customization

### Updating Content
All content is contained in `index.html`. To customize:

1. **Personal Information**: Update the hero section
```html
<h1>Your Name</h1>
<p class="subtitle">Your Title</p>
```

2. **Projects**: Add/modify project cards in the projects section
```html
<div class="project-card">
    <h3>Project Name</h3>
    <div class="date">Date Range</div>
    <ul>
        <li>Project bullet points</li>
    </ul>
</div>
```

3. **Experience**: Update timeline items in the experience section

4. **Skills**: Modify skill items in the skills grid

### Styling Changes
- **Colors**: Modify the color values in the CSS section
- **Fonts**: Change the `font-family` in the `body` style
- **Animation Speed**: Adjust `transition` and `animation` durations
- **3D Effects**: Customize particle count and geometry in the Three.js section
